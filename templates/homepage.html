Netflix Film Catalogue REST API

built by Nikita Izmailov

Available Endpoints HTTP Request Type
"/" GET method, displays available endpoints
"/api/films/id" GET method, for "id" provide a number like: 1,2,3,4...
"/api/films/id" PUT method, provide body parameter with JSON data to make changes to the specified film data
"/api/films/id" DELETE method, to delete a film with provided id
"/api/films/" POST method, provide body parameter with JSON data to create new film record
"/api/films?parameter_name=value¶meter_name=value" GET method, filter for specific film records
